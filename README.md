# boukincountweb

**棒金（端数を切り上げる慣習）を含めた金額計算ができる Web アプリ**です。  
日常的な割り勘や簡易的な会計を想定し、**シンプルで分かりやすいUI**を目指して作成しました。

🔗 **デモページ**  
https://shuta1123.github.io/boukincountweb/

---

## 📌 概要
このアプリは、入力された金額をもとに棒金処理を行い、  
実際に支払う金額を簡単に計算できる Web アプリです。

- ブラウザだけで動作
- インストール不要
- PC / スマートフォン対応（レスポンシブ）

---

## 🛠 使用技術
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **GitHub Pages**（デプロイ）


---

## ✨ 機能
- 金額入力
- 棒金処理を含めた計算
- 計算結果の即時表示
- スマートフォン表示対応（レスポンシブデザイン）

---

## 📱 レスポンシブ対応について
スマートフォン閲覧時でも操作しやすいよう、以下を意識して設計しました。

- 画面幅に応じたレイアウト調整
- 入力欄・ボタンの全幅表示
- 横スクロールが発生しない設計
